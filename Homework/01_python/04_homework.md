## 2. 가변 인자 리스트

### 가변 인자 리스트를 사용하여, 개수가 정해지지 않은 여러 정수들을 전달 받아 해당 정수들의 평균 값을 반환하는 my_avg 함수를 작성하시오.

- 문제

```python
my_avg(77,83,95,80,70) => 81.0
```

- 답안

```python
def my_avg(*args):
    avg = 0
    avg += sum(args)/len(args)
    print(avg)

my_avg(77,83,95,80,70)
```
